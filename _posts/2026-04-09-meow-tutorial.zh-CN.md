## 应用简介

喵窝记账是一款专为个人财务管理打造的可爱卡通风格记账应用。应用以猫咪为主题，界面设计萌趣可爱，同时功能强大且实用，帮助您轻松掌控个人财务状况，支持 Android 平台 [Google Play 下载](https://play.google.com/store/apps/details?id=com.glgjing.money.manager.bookkeeping.meow&hl=en) 和 iOS 平台 [App Store 下载](https://apps.apple.com/us/app/meow-money-manager/id1528852284)。

**🐱 超萌界面设计**
- 猫咪主题的可爱卡通界面，让记账变得有趣
- 500+ 个精心设计的卡通图标，每个图标都萌趣十足
- 支持深色模式、浅色模式和跟随系统主题，满足不同场景使用需求

**💰 全面的记账功能**
- 支持支出、收入、转账三种记账类型
- 丰富的分类图标，涵盖餐饮、购物、交通、娱乐、医疗等各类生活场景
- 支持添加备注，记录消费详情
- 快速记账，一键完成

**📊 智能预算管理**
- 支持多个预算同时追踪
- 灵活的预算周期设置：每日、每周、每月、每年或自定义周期
- 分类预算功能，为不同消费类别设置独立预算
- 实时显示预算使用情况、剩余金额和日均预算

**📈 强大的统计分析**
- 饼图展示：直观显示各类别收支占比
- 柱状图展示：按日或按月展示收支趋势
- 日历视图：通过颜色深浅快速识别每日收支情况
- 支持按日、周、月、年及自定义时间范围统计

**📒 多账本管理**
- 支持创建多个账本，分类管理不同类型的收支
- 每个账本独立统计，清晰明了
- 快速切换账本，方便记录不同场景的财务

**🔄 周期自动记账**
- 自动记录定期收支，如工资、房租、水电费等
- 支持每日、每周、每月、每年多种循环周期
- 支持同时选择多个日期，灵活设置记账时间

**💵 多货币支持**
- 支持50+种货币符号，包括人民币、美元、欧元、英镑等
- 满足不同国家和地区用户的需求

**🔒 隐私保护**
- 支持指纹应用锁，保护您的财务隐私
- 无需注册账号，数据完全保存在本地
- 不收集任何用户个人信息

**☁️ 数据备份与恢复**
- 支持 Google Drive 备份、iCloud 备份（iOS 版本）、WebDAV 云盘备份
- 自动备份功能，防止数据丢失
- 支持手动备份和恢复
- 数据导出为 CSV 格式，方便在 Excel 中查看和编辑

**💳 资产管理**
- 支持管理多种资产类型：现金、银行卡、信用卡、股票、基金等
- 资产趋势统计，掌握资产变化
- 资产间转账功能，方便资金调配

**🧾 报销管理**
- 专门用于管理需要报销的支出
- 追踪报销状态，区分已报销和未报销
- 支持部分报销和全额报销

## 记账功能

喵窝记账的记账功能非常直观，支持支出、收入和转账三种类型的记录。

### 添加|支出|收入|记录

1. 点击首页底部的记账按钮
2. 在弹出的记账界面中，切换到|支出|收入|标签
3. 选择合适的分类图标（如餐饮、购物、交通等）
4. 输入金额
5. 选择支付方式（资产）
6. 选择所属账本
7. 添加备注（可选）
8. 点击确认完成记账

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_ledger.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_assets.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_remark.jpg){: width="180" .normal }

### 转账功能

1. 点击首页底部的记账按钮
2. 在弹出的记账界面中，切换到转账标签
3. 选择转出资产和转入资产
4. 输入转账金额
5. 输入手续费（可选）
6. 选择所属账本
7. 添加备注（可选）
8. 点击确认完成转账

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_transfer.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_charge.jpg){: width="180" .normal }

## 预算管理

喵窝记账支持多种类型的预算管理，包括每日、每周、每月、每年和自定义周期的预算。

### 添加预算

1. 在主界面点击预算横幅或从设置中进入预算管理
2. 点击"添加预算"按钮
3. 选择预算类型（每日、每周、每月、每年或自定义）
4. 点击"添加分类预算"，选择要设置预算的分类
5. 输入总预算金额或者输入每个分类的预算金额
6. 输入预算名称
7. 点击确认完成预算创建

## 统计分析

## 常见问题

### 如何修改已添加的记账记录？

1. 在记账记录列表中，找到要修改的记录
2. 点击该记录，在弹出的页面中选择"编辑"
3. 修改相关信息
4. 点击确认完成修改

### 如何删除记账记录？

1. 在记账记录列表中，找到要删除的记录
2. 长按该记录，选择"删除"
3. 确认删除操作

### 如何切换货币符号？

1. 进入设置页面
2. 点击"货币符号设置"
3. 选择您需要的货币符号
4. 点击确认完成设置

### 如何设置应用锁？

1. 进入设置页面
2. 点击"隐私设置"
3. 开启"指纹锁"或"密码锁"
4. 按照提示完成设置

### 如何备份数据？

1. 进入设置页面
2. 点击"备份与恢复"
3. 选择备份方式
4. 点击"立即备份"进行手动备份
