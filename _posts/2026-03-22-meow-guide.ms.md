Urus kewangan peribadi dengan menyeronokkan. Meow Money Manager mempunyai antara muka tema kucing yang comel dan fungsi hebat. Ikuti 5 langkah mudah untuk menguasai aplikasi ini.

### Langkah 1: Muat turun & pasang Meow Money Manager
Muat turun dari [Google Play](https://play.google.com/store/apps/details?id=com.glgjing.money.manager.bookkeeping.meow) atau [App Store](https://apps.apple.com/us/app/meow-money-manager/id1528852284). Serasi dengan Android dan iOS.

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/other_google_play.png){: .normal }

### Langkah 2: Buat buku lejar pertama
Buka aplikasi dan tambah buku lejar baharu, namakannya seperti «Perbelanjaan Peribadi». Buka banyak buku untuk catat pendapatan dan perbelanjaan berasingan.

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/ledger_add.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/ledger_manage.jpg){: width="180" .normal }

### Langkah 3: Tetapkan belanjawan tersuai
Masuk halaman belanjawan, tetapkan had harian, mingguan atau bulanan. Tetapkan belanjawan mengikut kategori seperti makanan dan pengangkutan. Aplikasi beri amaran jika hampir had.

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/budget_manage.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/budget_add_period.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/budget_add_category.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/budget_add_amount.jpg){: width="180" .normal }

### Langkah 4: Catat pendapatan atau perbelanjaan pertama
Tekan butang «+», pilih jenis transaksi, ikon daripada 500+ kategori, masuk jumlah dan nota. Tetapkan catatan berulang untuk sewa dan gaji.

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add.jpg){: width="180" .normal }

### Langkah 5: Lihat statistik & lindungi privasi
Lihat carta pai, carta palang atau pandangan kalendar mengikut hari, bulan atau tahun. Aktifkan kunci cap jari atau kata laluan. Sandarkan ke iCloud/WebDAV dan eksport laporan CSV.
