Administra tus finanzas de forma divertida. Meow Money Manager tiene interfaz de gatos y herramientas potentes. Sigue estos 5 pasos.

![image](/assets/meow/page_home.jpg){: width="180" .normal } ![image](/assets/meow/page_dark_home.jpg){: width="180" .normal } ![image](/assets/meow/page_add_record.jpg){: width="180" .normal } ![image](/assets/meow/page_ledger_manage.jpg){: width="180" .normal }

### Paso 1: Descargar e instalar
Descarga desde [Google Play](https://play.google.com/store/apps/details?id=com.glgjing.money.manager.bookkeeping.meow) o [App Store](https://apps.apple.com/us/app/meow-money-manager/id1528852284). Compatible con Android e iOS.

![image](/assets/meow/page_google_play.png){: .normal }

### Paso 2: Crear tu primer libro de cuentas
Abre la app y crea un libro con nombre como «Gastos Personales». Agrega varios libros para separar finanzas.

![image](/assets/meow/page_home.jpg){: width="180" .normal } ![image](/assets/meow/page_ledger_add.jpg){: width="180" .normal } ![image](/assets/meow/page_ledger_manage.jpg){: width="180" .normal }

### Paso 3: Personalizar tu presupuesto
Configura presupuesto diario, semanal o mensual, por categorías (comida, transporte). La app alerta cerca del límite.

![image](/assets/meow/page_budget.jpg){: width="180" .normal } ![image](/assets/meow/page_budget_add_1.jpg){: width="180" .normal } ![image](/assets/meow/page_budget_add_2.jpg){: width="180" .normal } ![image](/assets/meow/page_budget_add_3.jpg){: width="180" .normal }

### Paso 4: Registrar ingreso o gasto
Toca «+», selecciona tipo, icono entre más de 500 categorías, monto y notas. Configura transacciones recurrentes para alquiler y salario.

![image](/assets/meow/page_add_record.jpg){: width="180" .normal }

### Paso 5: Ver estadísticas y proteger privacidad
Consulta gráficos y vista de calendario. Activa huella dactilar o contraseña. Copia de seguridad en iCloud/WebDAV y exporta CSV.

![image](/assets/meow/page_stats_1.jpg){: width="180" .normal } ![image](/assets/meow/page_stats_3.jpg){: width="180" .normal } ![image](/assets/meow/page_password_setting.jpg){: width="180" .normal } ![image](/assets/meow/page_password_enter.jpg){: width="180" .normal }

Muy sencillo. La app es gratuita; suscríbete para funciones premium. Descárgala hoy.
