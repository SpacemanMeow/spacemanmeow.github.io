จัดการการเงินไม่น่าเบื่ออีกต่อไป Meow Money Manager มีธีมแมวน่ารักและฟังก์ชันครบ ทำให้บันทึกการเงินง่ายและสนุก เพียง 5 ขั้นตอนก็ใช้งานได้เลย

![image](/assets/meow/page_home.jpg){: width="180" .normal } ![image](/assets/meow/page_dark_home.jpg){: width="180" .normal } ![image](/assets/meow/page_add_record.jpg){: width="180" .normal } ![image](/assets/meow/page_ledger_manage.jpg){: width="180" .normal }

### ขั้นตอน 1: ดาวน์โหลดและติดตั้ง
ดาวน์โหลดได้ที่ [Google Play](https://play.google.com/store/apps/details?id=com.glgjing.money.manager.bookkeeping.meow) หรือ [App Store](https://apps.apple.com/us/app/meow-money-manager/id1528852284) รองรับ Android และ iOS

![image](/assets/meow/page_google_play.png){: .normal }

### ขั้นตอน 2: สร้างสมุดบัญชีแรก
เปิดแอปแล้วสร้างสมุดบัญชี ตั้งชื่อ เช่น ค่าใช้จ่ายส่วนตัว สร้างหลายสมุดแยกตามประเภทได้

![image](/assets/meow/page_home.jpg){: width="180" .normal } ![image](/assets/meow/page_ledger_add.jpg){: width="180" .normal } ![image](/assets/meow/page_ledger_manage.jpg){: width="180" .normal }

### ขั้นตอน 3: ตั้งงบประมาณส่วนตัว
ตั้งงบรายวัน รายสัปดาห์ หรือรายเดือน และตั้งงบแยกหมวด อาหาร การเดินทาง แอปแจ้งเตือนเมื่อใกล้เกินงบ

![image](/assets/meow/page_budget.jpg){: width="180" .normal } ![image](/assets/meow/page_budget_add_1.jpg){: width="180" .normal } ![image](/assets/meow/page_budget_add_2.jpg){: width="180" .normal } ![image](/assets/meow/page_budget_add_3.jpg){: width="180" .normal }

### ขั้นตอน 4: บันทึกรายรับรายจ่ายแรก
กดปุ่ม «+» เลือกรายรับหรือรายจ่าย เลือกไอคอนจาก 500+ แบบ กรอกจำนวนเงินและหมายเหตุ ตั้งรายการซ้ำได้ เช่น เงินเดือน ค่าเช่า

![image](/assets/meow/page_add_record.jpg){: width="180" .normal }

### ขั้นตอน 5: ดูสถิติและปกป้องความเป็นส่วนตัว
ดูข้อมูลด้วยแผนภูมิวงกลม แท่ง หรือปฏิทิน เปิดล็อครหัสหรือลายนิ้วมือ สำรองข้อมูล iCloud/WebDAV และส่งออกไฟล์ CSV

![image](/assets/meow/page_stats_1.jpg){: width="180" .normal } ![image](/assets/meow/page_stats_3.jpg){: width="180" .normal } ![image](/assets/meow/page_password_setting.jpg){: width="180" .normal } ![image](/assets/meow/page_password_enter.jpg){: width="180" .normal }

ใช้งานง่ายมาก แอปใช้ฟรี สมัครสมาชิกปลดล็อกฟีเจอร์พรีเมียม ดาวน์โหลดวันนี้เลย
