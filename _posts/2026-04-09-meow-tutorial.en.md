## App Introduction

Meow Money Manager is a cute cartoon-style bookkeeping app designed for personal financial management. With a cat-themed interface, it combines cute design with powerful features to help you easily manage your personal finances. Available on both Android [Google Play Download](https://play.google.com/store/apps/details?id=com.glgjing.money.manager.bookkeeping.meow&hl=en) and iOS [App Store Download](https://apps.apple.com/us/app/meow-money-manager/id1528852284).

**🐱 Adorable Interface Design**
- Cat-themed cute cartoon interface that makes bookkeeping fun
- 500+ carefully designed cartoon icons, each one cute and interesting
- Supports dark mode, light mode, and system theme following, meeting different usage scenarios

**💰 Comprehensive Bookkeeping Features**
- Supports three transaction types: expenses, income, and transfers
- Rich category icons covering dining, shopping, transportation, entertainment, medical care, and other life scenarios
- Supports adding notes to record consumption details
- Quick bookkeeping, completed with one click

**📊 Smart Budget Management**
- Supports tracking multiple budgets simultaneously
- Flexible budget cycle settings: daily, weekly, monthly, yearly, or custom cycles
- Category budget feature, setting independent budgets for different consumption categories
- Real-time display of budget usage, remaining amount, and daily average budget

**📈 Powerful Statistics and Analysis**
- Pie chart display: intuitively shows the proportion of income and expenses by category
- Bar chart display: shows income and expenditure trends by day or month
- Calendar view: quickly identify daily income and expenditure through color depth
- Supports statistics by day, week, month, year, and custom time ranges

**📒 Multi-Ledger Management**
- Supports creating multiple ledgers to categorize different types of income and expenses
- Each ledger has independent statistics, clear and organized
- Quick ledger switching, convenient for recording finances in different scenarios

**🔄 Recurring Automatic Bookkeeping**
- Automatically records regular income and expenses, such as salary, rent, utilities, etc.
- Supports multiple cycle periods: daily, weekly, monthly, yearly
- Supports selecting multiple dates simultaneously, flexibly setting bookkeeping time

**💵 Multi-Currency Support**
- Supports 50+ currency symbols, including RMB, USD, EUR, GBP, etc.
- Meets the needs of users in different countries and regions

**🔒 Privacy Protection**
- Supports fingerprint app lock to protect your financial privacy
- No account registration required, data is completely stored locally
- Does not collect any user personal information

**☁️ Data Backup and Recovery**
- Supports Google Drive backup, iCloud backup (iOS version), and WebDAV cloud backup
- Automatic backup function to prevent data loss
- Supports manual backup and recovery
- Data export to CSV format, convenient for viewing and editing in Excel

**💳 Asset Management**
- Supports managing multiple asset types: cash, bank cards, credit cards, stocks, funds, etc.
- Asset trend statistics, tracking asset changes
- Asset transfer function, convenient for fund allocation

**🧾 Expense Reimbursement**
- Specifically designed for managing expenses that require reimbursement
- Tracks reimbursement status, distinguishing between reimbursed and unreimbursed
- Supports partial reimbursement and full reimbursement

## Bookkeeping Features

Meow Money Manager's bookkeeping features are very intuitive, supporting three types of records: expenses, income, and transfers.

### Add Expense or Income Record

1. Click the bookkeeping button at the bottom of the home page
2. In the pop-up bookkeeping interface, switch to the "Expense or Income" tab
3. Select an appropriate category icon (such as dining, shopping, transportation, etc.)
4. Enter the amount
5. Select payment method (asset)
6. Select the ledger
7. Add notes (optional)
8. Click confirm to complete bookkeeping

![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_ledger.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_assets.jpg){: width="180" .normal } ![image](https://cdn.jsdelivr.net/gh/spacemanmeow/spacemanmeow.github.io@main/assets/meow/record_add_remark.jpg){: width="180" .normal }

### Transfer Function

1. Click the bookkeeping button at the bottom of the home page
2. In the pop-up bookkeeping interface, switch to the transfer tab
3. Select the source asset and target asset
4. Enter the transfer amount
5. Enter handling fee (optional)
6. Select the ledger
7. Add notes (optional)
8. Click confirm to complete the transfer

## Budget Management

Meow Money Manager supports multiple types of budget management, including daily, weekly, monthly, yearly, and custom cycle budgets.

### Add Budget

1. On the main interface, click the budget banner or enter budget management from settings
2. Click the "Add Budget" button
3. Select budget type (daily, weekly, monthly, yearly, or custom)
4. Click "Add Category Budget" and select the categories for which you want to set budgets
5. Enter the total budget amount or enter the budget amount for each category
6. Enter budget name
7. Click confirm to complete budget creation

## Recurring Transactions

The recurring transaction feature can help you automatically record regular income and expenses, such as salary, rent, utilities, etc.

### Add Recurring Transaction

4. Select category
5. Enter amount
6. Select asset
7. Select ledger
8. Set cycle period (daily, weekly, monthly, yearly)
9. Set start date and end date

## FAQ

### How to modify an added bookkeeping record?

1. In the bookkeeping record list, find the record you want to modify
2. Click on the record and select "Edit" in the pop-up page
3. Modify relevant information
4. Click confirm to complete the modification

### How to delete a bookkeeping record?

1. In the bookkeeping record list, find the record you want to delete
2. Long press on the record and select "Delete"
3. Confirm the deletion operation

### How to switch currency symbols?

1. Enter the settings page
2. Click "Currency Symbol Settings"
3. Select the currency symbol you need
4. Click confirm to complete the setting

### How to set up app lock?

1. Enter the settings page
2. Click "Privacy Settings"
3. Enable "Fingerprint Lock" or "Password Lock"
4. Follow the prompts to complete the setup

### How to back up data?

1. Enter the settings page
2. Click "Backup and Restore"
3. Select backup method
4. Click "Backup Now" for manual backup
