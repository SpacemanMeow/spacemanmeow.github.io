Представление приложения

Meow Money Manager — это милое бухгалтерское приложение в стиле мультфильма, разработанное для личного финансового менеджмента. С интерфейсом с тематикой котов, оно сочетает милый дизайн с мощными функциями, чтобы помочь вам легко управлять личными финансами. Доступно на Android Скачать из Google Play и iOS Скачать из App Store.

Первые шаги

Основной интерфейс включает следующие основные части:

  • Кнопка бухгалтерии: расположена в правом нижнем углу, используется для быстрого добавления записей бухгалтерии
  • Нижняя панель навигации: включает четыре опции: бухгалтерия, статистика, активы и настройки
  • Верхний баннер: отображает текущую книгу и сводку активов

Функции бухгалтерии

Функции бухгалтерии Meow Money Manager очень интуитивны, поддерживают три типа записей: расходы, доходы и переводы.

Добавить | Расходы | Доходы | Записи

  1. Нажмите кнопку бухгалтерии в нижней части главной страницы
  2. В всплывающем интерфейсе бухгалтерии перейдите на вкладку | Расходы | Доходы |
  3. Выберите соответствующую иконку категории (например, питание, покупки, транспорт и т.д.)
  4. Введите сумму
  5. Выберите метод оплаты (актив)
  6. Выберите книгу
  7. Добавьте заметку (необязательно)
  8. Нажмите подтвердить, чтобы завершить бухгалтерию

Функция перевода

  1. Нажмите кнопку бухгалтерии в нижней части главной страницы
  2. В всплывающем интерфейсе бухгалтерии перейдите на вкладку перевода
  3. Выберите исходный актив и целевой актив
  4. Введите сумму перевода
  5. Введите комиссию за обработку (необязательно)
  6. Выберите книгу
  7. Добавьте заметку (необязательно)
  8. Нажмите подтвердить, чтобы завершить перевод

Управление бюджетом

Meow Money Manager поддерживает различные типы управления бюджетом, включая дневной, еженедельный, ежемесячный, ежегодный бюджет и пользовательский цикл.

Добавить бюджет

  1. В основном интерфейсе нажмите баннер бюджета или перейдите в управление бюджетом из настроек
  2. Нажмите кнопку "Добавить бюджет"
  3. Выберите тип бюджета ( дневной, еженедельный, ежемесячный, ежегодный или пользовательский)
  4. Нажмите "Добавить бюджет по категориям" и выберите категорию, для которой вы хотите установить бюджет
  5. Введите общую сумму бюджета или введите сумму бюджета для каждой категории
  6. Введите название бюджета
  7. Нажмите подтвердить, чтобы завершить создание бюджета

Отслеживание бюджета

На странице деталей бюджета вы можете увидеть:

  • Общий бюджет и использованная сумма
  • Остаток бюджета и оставшиеся дни
  • Средний дневной бюджет
  • Использование бюджета по категориям

Статистика и анализ

Meow Money Manager предоставляет различные функции статистического анализа, чтобы помочь вам понять ваше финансовое состояние.

Типы статистики

  1. В основном интерфейсе нажмите нижнюю панель навигации статистики, чтобы перейти на страницу статистики
  2. Выберите временной интервал статистики: день, неделя, месяц, год или пользовательский
  3. Просмотрите различные типы статистических диаграмм
  • Статистика круговой диаграммы : Круговая диаграмма показывает долю расходов и доходов в различных категориях, помогая вам понять, где тратятся ваши деньги.
  • Статистика столбчатой диаграммы : Столбчатая диаграмма показывает дневные или месячные доходы и расходы, помогая вам понять тенденции доходов и расходов.
  • Календарный просмотр : Календарный просмотр показывает дневные доходы и расходы, с глубиной цвета, указывающей размер суммы.

Управление активами

Meow Money Manager поддерживает управление различными типами активов, такими как наличные деньги, банковские карты, кредитные карты и т.д.

Добавить актив

  1. В основном интерфейсе нажмите нижнюю панель навигации активов, чтобы перейти на страницу активов
  2. Нажмите кнопку "Добавить актив"
  3. Выберите тип актива
  4. Введите название актива
  5. Введите начальную сумму
  6. Выберите иконку актива
  7. Нажмите подтвердить, чтобы завершить добавление актива

Детали актива

Нажмите на актив, чтобы перейти на страницу деталей, где вы можете увидеть:

  • Баланс актива
  • Записи доходов и расходов актива
  • Диаграмма тенденций актива

Управление книгами

Meow Money Manager поддерживает создание нескольких книг для управления различными типами доходов и расходов.

Добавить книгу

  1. В основном интерфейсе нажмите баннер книги или перейдите в управление книгами из настроек
  2. Нажмите кнопку "Добавить книгу"
  3. Введите название книги
  4. Выберите иконку книги
  5. Нажмите подтвердить, чтобы завершить создание книги

Сменить книгу

  1. При выполнении бухгалтерии нажмите селектор книги
  2. Выберите книгу, которую вы хотите использовать, из всплывающего списка книг
  3. После завершения выбора будут отображаться записи бухгалтерии для этой книги

Статистика книги

Нажмите на книгу, чтобы перейти на страницу деталей, где вы можете увидеть для этой книги:

  • Статистика доходов и расходов
  • Использование бюджета
  • Подробные записи бухгалтерии

Повторяющиеся транзакции

Функция повторяющихся транзакций может помочь вам автоматически записывать регулярные доходы и расходы, такие как зарплата, аренда, коммунальные услуги и т.д.

Добавить повторяющуюся транзакцию

  1. Перейдите в управление повторяющимися транзакциями из настроек
  2. Нажмите кнопку "Добавить повторяющуюся транзакцию"
  3. Выберите тип транзакции (расход или доход)
  4. Выберите категорию
  5. Введите сумму
  6. Выберите актив
  7. Выберите книгу
  8. Установите период цикла ( дневной, еженедельный, ежемесячный, ежегодный)
  9. Установите даты начала и окончания
  10. Нажмите подтвердить, чтобы завершить настройку повторяющейся транзакции

Управление повторяющимися транзакциями

В списке повторяющихся транзакций вы можете:

  • Просмотреть все задачи повторяющихся транзакций
  • Редактировать или удалить задачи повторяющихся транзакций
  • Вручную запустить повторяющиеся транзакции

Возмещение расходов

Функция возмещения расходов может помочь вам отслеживать расходы, требующие возмещения.

Добавить запись о возмещении

  1. При добавлении записи о расходе отметьте опцию "Возмещение"
  2. Заполните другую информацию о бухгалтерии
  3. Нажмите подтвердить, чтобы завершить добавление записи о возмещении

Управление возмещением

  1. Перейдите в управление возмещением из настроек
  2. Просмотрите все ожидающие и возмещенные записи
  3. Нажмите запись о возмещении и отметьте ее как возмещенную
  4. Введите сумму возмещения и дату возмещения
  5. Выберите метод возмещения (актив)
  6. Нажмите подтвердить, чтобы завершить возмещение

Настройки

Meow Money Manager предоставляет различные параметры настройки, чтобы помочь вам настроить функции приложения.

Основные настройки

  1. Перейдите на страницу настроек из нижней панели навигации
  2. Отрегулируйте следующие настройки:
- Настройка темы (светлая, темная или следование за системой) - Настройка языка - Настройка символа валюты - Настройка первого дня недели - Настройка первого дня месяца

Настройки конфиденциальности

  1. На странице настроек нажмите "Настройки конфиденциальности"
  2. Включите или отключите блокировку приложения отпечатком пальца
  3. Установите пароль приложения (если необходимо)

Резервное копирование и восстановление

  1. На странице настроек нажмите "Резервное копирование и восстановление"
  2. Выберите метод резервного копирования:
- Google Drive - Резервное копирование iCloud (только iOS) - Резервное копирование WebDAV
  1. Нажмите "Сделать резервную копию сейчас" для выполнения ручного резервного копирования
  2. Нажмите "Восстановить" для восстановления данных из файла резервной копии

Экспорт данных

  1. На странице настроек нажмите "Экспорт данных"
  2. Выберите временной интервал экспорта
  3. Выберите формат экспорта (CSV)
  4. Нажмите "Экспортировать" для экспорта данных в файл

Часто задаваемые вопросы

Как изменить запись бухгалтерии, которая уже была добавлена?

  1. В списке записей бухгалтерии найдите запись, которую вы хотите изменить
  2. Нажмите на запись и выберите "Редактировать" на всплывающей странице
  3. Измените соответствующую информацию
  4. Нажмите подтвердить, чтобы завершить изменения

Как удалить запись бухгалтерии?

  1. В списке записей бухгалтерии найдите запись, которую вы хотите удалить
  2. Держите запись нажатой и выберите "Удалить"
  3. Подтвердите операцию удаления

Как изменить символ валюты?

  1. Перейдите на страницу настроек
  2. Нажмите "Настройка символа валюты"
  3. Выберите символ валюты, который вам нужен
  4. Нажмите подтвердить, чтобы завершить настройку

Как установить блокировку приложения?

  1. Перейдите на страницу настроек
  2. Нажмите "Настройки конфиденциальности"
  3. Включите "Блокировка отпечатком пальца" или "Блокировка паролем"
  4. Следуйте инструкциям для завершения настройки

Как сделать резервную копию данных?

  1. Перейдите на страницу настроек
  2. Нажмите "Резервное копирование и восстановление"
  3. Выберите метод резервного копирования
  4. Нажмите "Сделать резервную копию сейчас" для выполнения ручного резервного копирования
