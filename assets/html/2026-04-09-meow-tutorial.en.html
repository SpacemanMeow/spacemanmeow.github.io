First-time Use

The main interface includes the following main parts:

  • Bookkeeping button: located at the bottom right, used to quickly add bookkeeping records
  • Bottom navigation bar: includes four options: bookkeeping, statistics, assets, and settings
  • Top banner: displays current ledger and asset overview

Bookkeeping Features

Meow Money Manager's bookkeeping features are very intuitive, supporting three types of records: expenses, income, and transfers.

Add | Expense | Income | Record

  1. Click the bookkeeping button at the bottom of the home page
  2. In the pop-up bookkeeping interface, switch to the | Expense | Income | tab
  3. Select an appropriate category icon (such as dining, shopping, transportation, etc.)
  4. Enter the amount
  5. Select payment method (asset)
  6. Select the ledger
  7. Add notes (optional)
  8. Click confirm to complete bookkeeping

Transfer Function

  1. Click the bookkeeping button at the bottom of the home page
  2. In the pop-up bookkeeping interface, switch to the transfer tab
  3. Select the source asset and target asset
  4. Enter the transfer amount
  5. Enter handling fee (optional)
  6. Select the ledger
  7. Add notes (optional)
  8. Click confirm to complete the transfer

Budget Management

Meow Money Manager supports multiple types of budget management, including daily, weekly, monthly, yearly, and custom cycle budgets.

Add Budget

  1. On the main interface, click the budget banner or enter budget management from settings
  2. Click the "Add Budget" button
  3. Select budget type (daily, weekly, monthly, yearly, or custom)
  4. Click "Add Category Budget" and select the categories for which you want to set budgets
  5. Enter the total budget amount or enter the budget amount for each category
  6. Enter budget name
  7. Click confirm to complete budget creation

Budget Tracking

On the budget details page, you can view:

  • Total budget and used amount
  • Remaining budget and remaining days
  • Daily average budget
  • Budget usage for each category

Statistics and Analysis

Meow Money Manager provides multiple statistical analysis functions to help you understand your financial situation.

Statistics Types

  1. On the main interface, click the bottom statistics navigation bar to enter the statistics page
  2. Select the statistical time range: day, week, month, year, or custom
  3. View different types of statistical charts
  • Pie chart statistics: Pie chart shows the proportion of expenses and income in different categories, helping you understand where your money is spent.
  • Bar chart statistics: Bar chart shows daily or monthly income and expenditure, helping you understand income and expenditure trends.
  • Calendar view: Calendar view shows daily income and expenditure, with color depth indicating amount size.

Asset Management

Meow Money Manager supports managing multiple types of assets, such as cash, bank cards, credit cards, etc.

Add Asset

  1. On the main interface, click the bottom asset navigation bar to enter the asset page
  2. Click the "Add Asset" button
  3. Select asset type
  4. Enter asset name
  5. Enter initial amount
  6. Select asset icon
  7. Click confirm to complete asset addition

Asset Details

Click on an asset to enter the details page, where you can view:

  • Asset balance
  • Asset income and expenditure records
  • Asset trend charts

Ledger Management

Meow Money Manager supports creating multiple ledgers for managing different types of income and expenses.

Add Ledger

  1. On the main interface, click the ledger banner or enter ledger management from settings
  2. Click the "Add Ledger" button
  3. Enter ledger name
  4. Select ledger icon
  5. Click confirm to complete ledger creation

Switch Ledger

  1. When bookkeeping, click the ledger selector
  2. Select the ledger to use from the pop-up ledger list
  3. After completing the selection, the bookkeeping records for that ledger will be displayed

Ledger Statistics

Click on a ledger to enter the details page, where you can view for that ledger:

  • Income and expenditure statistics
  • Budget usage
  • Detailed bookkeeping records

Recurring Transactions

The recurring transaction feature can help you automatically record regular income and expenses, such as salary, rent, utilities, etc.

Add Recurring Transaction

  1. Enter recurring transaction management from settings
  2. Click the "Add Recurring Transaction" button
  3. Select transaction type (expense or income)
  4. Select category
  5. Enter amount
  6. Select asset
  7. Select ledger
  8. Set cycle period (daily, weekly, monthly, yearly)
  9. Set start date and end date
  10. Click confirm to complete recurring transaction setup

Recurring Transaction Management

In the recurring transaction list, you can:

  • View all recurring transaction tasks
  • Edit or delete recurring transaction tasks
  • Manually trigger recurring transactions

Expense Reimbursement

The expense reimbursement feature can help you track expenses that need to be reimbursed.

Add Reimbursement Record

  1. When adding an expense record, check the "Reimburse" option
  2. Complete other bookkeeping information
  3. Click confirm to complete reimbursement record addition

Reimbursement Management

  1. Enter reimbursement management from settings
  2. View all pending and reimbursed records
  3. Click on a reimbursement record and mark it as reimbursed
  4. Enter reimbursement amount and reimbursement date
  5. Select reimbursement method (asset)
  6. Click confirm to complete reimbursement

Settings

Meow Money Manager provides multiple setting options to help you customize app functions.

Basic Settings

  1. Enter the settings page from the bottom navigation bar
  2. Adjust the following settings:
- Theme settings (light, dark, or follow system) - Language settings - Currency symbol settings - Week start day settings - Month start day settings

Privacy Settings

  1. On the settings page, click "Privacy Settings"
  2. Enable or disable fingerprint app lock
  3. Set app password (if needed)

Backup and Restore

  1. On the settings page, click "Backup and Restore"
  2. Select backup method:
- Google Drive - iCloud backup (iOS only) - WebDAV backup
  1. Click "Backup Now" for manual backup
  2. Click "Restore" to restore data from backup files

Export Data

  1. On the settings page, click "Export Data"
  2. Select export time range
  3. Select export format (CSV)
  4. Click "Export" to export data to a file

FAQ

How to modify an added bookkeeping record?

  1. In the bookkeeping record list, find the record you want to modify
  2. Click on the record and select "Edit" in the pop-up page
  3. Modify relevant information
  4. Click confirm to complete the modification

How to delete a bookkeeping record?

  1. In the bookkeeping record list, find the record you want to delete
  2. Long press on the record and select "Delete"
  3. Confirm the deletion operation

How to switch currency symbols?

  1. Enter the settings page
  2. Click "Currency Symbol Settings"
  3. Select the currency symbol you need
  4. Click confirm to complete the setting

How to set up app lock?

  1. Enter the settings page
  2. Click "Privacy Settings"
  3. Enable "Fingerprint Lock" or "Password Lock"
  4. Follow the prompts to complete the setup

How to back up data?

  1. Enter the settings page
  2. Click "Backup and Restore"
  3. Select backup method
  4. Click "Backup Now" for manual backup
